3. Vai in Build → Realtime Database → crea database → scegli Test mode
4. Vai in Build → Storage → inizializza → scegli Test mode
5. Torna in Impostazioni progetto e copia la configurazione qui sotto:

1. Vai su console.cloud.google.com (accedi con Google)

5. Tipo applicazione: Applicazione web
6. Aggiungi in "Origini JavaScript autorizzate": https://tradocroods.netlify.app
